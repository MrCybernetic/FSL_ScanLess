# SCANLESS 

![EN Flag](https://github.com/MrCybernetic/FSL_ScanLess/blob/master/img/en.png) EN

Designing an open source PCB to turn your Freestyle Libre into a wireless CGM (continuous glucose monitor) based on LimiTTer ([Limitter GitHub](https://github.com/JoernL/LimiTTer)).

--- 

![FR Flag](https://github.com/MrCybernetic/FSL_ScanLess/blob/master/img/fr.png) FR

Le but de ce projet est de concevoir un PCB en open source pour suivre votre glycémie via votre smartphone. Ce projet est basé sur celui-ci : LimiTTer ([GitHub Limitter](https://github.com/JoernL/LimiTTer)).
